# Privacy Policy for PresentationPointer

**Last updated: 2026-07-23**

## Overview

PresentationPointer is a presentation pointer tool for Windows.
This app does not collect, store, transmit, or share any personal information or user data.

## Data Collection

This app collects **no data** of any kind, including but not limited to:

- Personal information
- Usage statistics or analytics
- Device information
- Location data
- Files or clipboard contents

## Network Access

Although the `internetClient` capability is declared in the app package manifest,
the app does **not** make any network connections.
This declaration is a default included during packaging and is not actively used.

## Third-Party Services

This app does not use any third-party services, SDKs, or advertising frameworks.

## Changes to This Policy

Any future updates to this privacy policy will be reflected in this document.
